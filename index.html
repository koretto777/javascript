Javascript

Es un lenguaje de programación dinámico y de alto nivel para crear interfaces interactivas y dinámicas.

Es uno de los 3 pilares fundamentales del desarrollo web.

  • HTML -> Define la estructura del contenido (elementos que aparecen en la página)
  • CSS -> Define la apariencia (colores, tamaños,estilos)
  • Javascript -> Define el comportamiento (lo que hace la página o sus elementos)

Características

  1. Lenguaje interpretado -> No necesitamos compilarlo. El navegador lo entiende directamente.
  2. Basado en objetos -> Maneja en objetos como document, window, arrays, dates, class.
  3. Dinámico -> Podemos modificar el contenido en tiempo real. (sin recargar)
  4. Lenguaje versatil y de fácil aprendizaje -> Aunque nacio para la web, se usa tmb en servidores, aplicaciones móviles, videojuegos y hoy en día en IA
